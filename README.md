# 🔐 AI Phishing Detection Chrome Extension

A real-time phishing detection system that checks whether a website is **SAFE** or **PHISHING** using a browser extension and Flask backend.

---

## 🚀 Features

* 🔍 Real-time URL scanning
* 🌐 Chrome extension integration
* ⚡ Fast Flask API
* 🔒 Handles CORS & security issues

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python, Flask
* **Tools:** Git, GitHub

---

## ▶️ How to Run

1. Install dependencies:

   ```
   pip install flask flask-cors
   ```
2. Run server:

   ```
   python app.py
   ```
3. Load extension in `chrome://extensions/`
4. Click **Scan** to check website

---

## 🧠 How It Works

Extension → Sends URL → Flask API → Returns SAFE/PHISHING → Displays result
